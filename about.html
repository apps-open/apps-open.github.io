Now OpenApps is developing free and open source applications for Linux distributions. You can help by submitting any application ideas (contacts will be available soon). OpenApps is also in need of financial assistance (see the "Donations" page).

Privacy

OpenApps respects user privacy, which is why it uses open source tools and APIs to develop it. In addition, OpenApps does NOT collect any personal data such as IP addresses, user agents, etc. To avoid many risks, OpenApps asks you to only download your applications from this site or GitHub repository.
